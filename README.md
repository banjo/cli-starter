# pkg-name

[![NPM version](https://img.shields.io/npm/v/pkg-name?color=%23c53635&label=%20)](https://www.npmjs.com/package/pkg-name)

A simple CLI application template.

## Install

```bash
npm install pkg-name -g
```

## Usage

```bash
# run
pkg-name

# run subcommand
pkg-name subcommand <name>
```
